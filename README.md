## 📝 О проекте

Веб-приложение интернет-магазина электроники с системой аутентификации, авторизации и управления доступом на основе ролей (RBAC). Разработано в рамках практических занятий №7–11.

В ходе контрольной работы были изучены и применены следующие технологии и концепции:

- **Хэширование паролей** (практическая работа №7) — безопасное хранение паролей с помощью bcrypt
- **JWT** (практическая работа №8) — аутентификация пользователей с помощью JSON Web Tokens
- **Access и Refresh токены** (практические работы №9–10) — разделение токенов, ротация refresh-токенов, автоматическое обновление access-токена через interceptors
- **Роли и права доступа** (практическая работа №11) — система RBAC с ролями User, Seller, Admin

## 🔐 Роли и права доступа

| Маршрут | Метод | Гость | User | Seller | Admin |
|---|---|---|---|---|---|
| /api/auth/register | POST | ✅ | ✅ | ✅ | ✅ |
| /api/auth/login | POST | ✅ | ✅ | ✅ | ✅ |
| /api/auth/refresh | POST | ✅ | ✅ | ✅ | ✅ |
| /api/auth/me | GET | ❌ | ✅ | ✅ | ✅ |
| /api/users | GET | ❌ | ❌ | ❌ | ✅ |
| /api/users/:id | GET/PUT/DELETE | ❌ | ❌ | ❌ | ✅ |
| /api/products | GET | ✅ | ✅ | ✅ | ✅ |
| /api/products | POST | ❌ | ❌ | ✅ | ✅ |
| /api/products/:id | GET | ❌ | ✅ | ✅ | ✅ |
| /api/products/:id | PUT | ❌ | ❌ | ✅ | ✅ |
| /api/products/:id | DELETE | ❌ | ❌ | ❌ | ✅ |

**Описание ролей:**
- **Гость** — не аутентифицированный пользователь, только просмотр каталога
- **User** — зарегистрированный пользователь, просмотр товаров
- **Seller** — продавец, создание и редактирование товаров
- **Admin** — администратор, полный доступ включая управление пользователями

## 📦 Установка и запуск

### Бэкенд
```bash
cd backend
npm install
node app.js
```
Сервер запустится на http://localhost:3000  
Swagger UI: http://localhost:3000/api-docs

### Фронтенд
```bash
cd frontend
npm install
npm run dev
```
Приложение откроется на http://localhost:5173

## 🌐 API Endpoints

### Аутентификация
| Метод | Endpoint | Описание |
|---|---|---|
| POST | /api/auth/register | Регистрация пользователя |
| POST | /api/auth/login | Вход в систему |
| POST | /api/auth/refresh | Обновление пары токенов |
| GET | /api/auth/me | Получение текущего пользователя |

### Пользователи (только Admin)
| Метод | Endpoint | Описание |
|---|---|---|
| GET | /api/users | Список всех пользователей |
| GET | /api/users/:id | Пользователь по ID |
| PUT | /api/users/:id | Обновление пользователя / смена роли |
| DELETE | /api/users/:id | Блокировка пользователя |

### Товары
| Метод | Endpoint | Описание |
|---|---|---|
| GET | /api/products | Список товаров |
| GET | /api/products/:id | Товар по ID |
| POST | /api/products | Создать товар |
| PUT | /api/products/:id | Обновить товар |
| DELETE | /api/products/:id | Удалить товар |
